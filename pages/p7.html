Data Normalization

Normalization is a technique often applied as part of data preparation for machine learning. The goal of
normalization is to change the values of numeric columns in the dataset to use a common scale, without
distorting differences in the ranges of values or losing information. Normalization is also required for
some algorithms to model the data correctly.

For example, assume your input dataset contains one column with values ranging from 0 to 1, and another
column with values ranging from 10,000 to 100,000. The great difference in the scale of the numbers could
cause problems when you attempt to combine the values as features during modeling.

Normalization avoids these problems by creating new values that maintain the general distribution and ratios
in the source data, while keeping values within a scale applied across all numeric columns used in the
model.
